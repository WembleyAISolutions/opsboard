# Docs Baseline

This folder tracks the intentionally small v0.1 definition for OpsBoard.

## Included in v0.1

- App scaffold with Next.js, React, TypeScript, Tailwind, App Router
- Quiet Desktop Lite UI skeleton
- Core module definitions and local mock data

## Excluded from v0.1

- Backend and APIs
- Database and persistence
- Authentication
- Agent orchestration and workflow engines
- Enterprise features
- Full task board or chat product behavior
