# OpsBoard v0.1 Scope Lock

## Product definition

OpsBoard is a lightweight human-AI work gateway for ChatGPT-centered multi-agent workflows.

## Core modules

1. Inbox
2. Signal
3. Approvals
4. Voice Capture
5. Today

## Build constraints

- Frontend only
- Local mock data only
- Desktop Lite presentation
- Minimal types and schema surface

## Out of scope

- Backend services
- Databases
- Auth and user management
- Agent orchestration
- Enterprise controls or integrations
- Conversion into a task board or chat app
